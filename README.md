# Pokeshop

![Pokemon Logo](https://raw.githubusercontent.com/MFRamon/Pokeshop/master/pokemon-logo.png)

This is a showcase project developed in React.JS

The tech stack used is: 

* React.JS 
* Enzyme 
* Jest
* Circle CI
* ESLint
* Styled Components

Future Features to implement:
